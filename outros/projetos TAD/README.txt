Código desenvolvido durante sessões de estudos feitas no meu tempo livre. 


Vector2f: visa permitir criar vetores de duas dimensões (x e y) e viabilizar operações comuns com vetores.
Tais como soma, multiplicação por um escalar, entre outros. Criei duas versões desse código, sendo a original feita em C++ e utilizando o conceito de orientação a objetos, e essa que está presente nessa pasta, desenvolvida em C como uma API TAD - Tipos abstratos de dados.
