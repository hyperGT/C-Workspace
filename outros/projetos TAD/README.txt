Código desenvolvido durante sessões de estudos feitas no meu tempo livre. 


Vector2f: permite a criação de vetores de duas dimensões (x, y) e viabiliza operações com vetores,
tais como soma, multiplicação e outros. Criei duas versões desse código, sendo a original feita em C++ e utilizando o conceito de orientação a objetos, e essa que está presente nessa pasta, desenvolvida em C como uma API TAD - Tipos abstratos de dados.
